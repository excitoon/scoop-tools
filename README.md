# `scoop-tools`

Scoop tools.

## `scoop-rehash`

Updates hash of source file in package.

### Dependencies

- Python 3

## `scoop-reshim`

Updates shims in local Scoop installation.

### Dependencies

```
pip3 install requests
```

## `scoop-validate`

Validates packages against schema.

### Dependencies

```
pip3 install jsonschema
pip3 install requests
```
